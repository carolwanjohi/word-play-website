# Word Puzzle Website

## A website that replaces vowels in a string with a dash symbol, 16/08/2017.

## By **[John Mutavi](https://github.com/jonnygovish) and [Carol Wanjohi](https://carolwanjohi.github.io/)**

## Description
This is a website that replaces vowels in a string with a dash symbol to create a word puzzle. The aim is to practise looping with for.
